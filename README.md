## 《玉书坊》 小说阅读

- 项目主要业务功能
  - 首页展示推荐以及轮播  (已实现)
  - 拉取指定文章的详情以及列表 (已实现)
  - 键入关键字搜索符合条件的书籍列表 (已实现)
  - 展示指定文章的内容,和切换下一章内容 (已实现)
  - 文章分类 (已实现)
  - 阅读历史记录 (待实现)
  - 注册登录,用户登录权限验证 (待实现)
  - 内容阅读页面背景切换,字体大小调整 (已实现)
